position

  • left, top, right, bottom, z-index 속성을 추가적으로 사용할 수 있음
  • z-index : 0, 큰 숫자는 위로 배치, 작은 숫자는 아래로 배치
  • static : default, position 해제시
  • fixed : 브라우저를 기준으로 배치, top 버튼
  • relative : absolute를 자식으로 가진 부모
  • absolute : 부모의 요소를 기준으로 배치할 때 설정
  • sticky : 최근에 사용되는 레이아웃
