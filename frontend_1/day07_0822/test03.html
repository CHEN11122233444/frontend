box model

block element(요소) 
        box-model : 아래와 같은 관계가 있음

        padding : 안쪽 여백, 배경색칠
        border : 테두리의 두께도 너비에 포함됨
        margin : 바깥쪽 여백
        width, height : 내용(content)
        실제 너비 : padding(left, right) + margin(left, right) + border(left, right) + width
        실제 높이 : padding(top, bottom) + margin(top, bottom) + border(top, bottom) + height
    
        box-sizing : border-box;
        실제 너비 : width(padding, border) + margin
        실제 높이 : height(padding, border) + margin
